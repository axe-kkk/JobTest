main.py
parser.py
bot_modul.py
requirements.txt

2. Створіть та активуйте віртуальне середовище:

python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

3. Встановіть необхідні бібліотеки:

pip install -r requirements.txt

4. Запустіть основний файл:

python main.py
